🔹 Churn-Prediction-Model
Customer churn prediction is a critical business problem where the objective is to identify customers who are likely to discontinue a service. This project implements an end-to-end machine learning pipeline to predict customer churn using structured customer data and advanced ensemble learning techniques.
The solution focuses on handling real-world challenges such as class imbalance, feature encoding, and model evaluation, making it suitable for practical business applications.

🔹 Model Development
The following models were trained and evaluated:
(a) Logistic Regression (Baseline model)
(b) Random Forest Classifier
(c) XGBoost Classifier (Final Model)

🔹Technologies Used
(a) Python, Pandas, NumPy, Scikit-learn, Imbalanced-learn (SMOTE), Matplotlib, Seaborn

🔹I handled class imbalance using SMOTE, compared Logistic Regression, Random Forest, and XGBoost, and finalized XGBoost due to its superior recall and ROC–AUC performance.
